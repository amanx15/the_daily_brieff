📋 Daily Standard Operating Procedure

The full step-by-step routine for using this tool effectively, end-to-end. Follow this once a day (typically 3-5pm MT, end of your shift) to keep your data, reports, and manager updates current.

1

Start your shift — open the dashboard

Open the_daily_brief.html in your browser. Check the masthead — you should see today's date in MT (Mountain Time, your manager's TZ) and IST (your local).

  • If it's a new month, switch the Period picker to the current month
  • Verify your work timezone is MT (default)
  • Scan the Overview hero metrics for anything unusual since yesterday
  • Check the per-brand mini-dashboards at the bottom of Overview — each brand's pace, forecast, lagging products, and CTA at a glance
2

Check anomaly alerts

If the red/amber anomaly banner appears on Overview, read it carefully. Common alerts:

  • Pace alert — you're 30%+ behind monthly goal. Open Insights tab for the catch-up plan.
  • Brand stalled — one brand has zero approvals while others are progressing. Time to investigate.
  • OOS blocking target — chase restock ETAs from the brand team.
  • Sudden drop — numbers went down vs last save; usually a data-entry mistake.
3

Log today's sample approvals (Input tab)

Press 2 to jump to Input. Three ways to enter data:

  • 📎 Upload CSV per brand — easiest. Export the order CSV from TikTok Shop Seller Center for each brand, click the 📎 Upload CSV button on the brand card, and it auto-counts Shipped orders per product. Recommended for daily use.
  • Form mode — type approved counts directly per product. Useful for quick edits. Sum expressions like 355+2019+37609 auto-evaluate.
  • Paste & Parse — paste raw shift notes and the dashboard auto-fills.
4

Log your daily tasks

Below the brand cards, fill in your task counts for each brand:

  • Outreach bot target plans created today
  • Segment bot messages sent
  • Ads codes authorized (UDC / MT) or qualified codes (Snackworks)
  • URL processing (UDC), affiliate drafts reviewed (Snackworks)
  • For Snackworks paired tasks: log both reviewed and qualified — the dashboard auto-calculates the rate
5

Add today's notes

In the Notes box (still on Input tab), capture anything important:

  • Wins worth flagging to your manager
  • Blockers (restock ETAs, creator issues, platform bugs)
  • Decisions made today
  • Tomorrow's plan

These flow into the generated report and the Executive Summary automatically.

6

Generate the daily report

Press G (or click Generate Report). This produces a brand-grouped, DM-ready report on the Report tab. Press C on that tab to copy to clipboard.

7

Save to history

Press S (or click Save to History). This is critical — it powers:

  • The Month-over-Month comparison tile
  • The trend chart on Overview
  • The anomaly detection system (detects drops between saves)
  • The History tab for looking up any past day

If you skip saving, you lose yesterday's snapshot when you log today's data.

8

Generate an Executive Summary for your manager

Back on the Overview tab, click 📤 Generate Executive Summary. A modal opens with a polished snapshot: headline, forecast, by-brand breakdown, top wins, top risks, asks, and notes.

  • Click Copy as Slack/Lark for daily team posts
  • Click Copy as Email for weekly skip-level updates
  • Click Print for paper review meetings
9

Closing check with the assistant

Press A to summon the ghost assistant. Try asking:

  • "What should I prioritize tomorrow?"
  • "Will I hit my goal at this pace?"
  • "What's behind today?"
  • "How does this compare to last month?"

The assistant reads your live data and gives specific, contextual answers.

Pro tip: Save the Campaign Vault (tab 9) with your standard outreach copy, important PIDs, restock contacts, and campaign briefs. The assistant can fetch anything from there when you need it during your shift.

Common questions (FAQ)
How does the Forecast tile work?

It takes your daily average (total approved ÷ days elapsed) and projects across the full month. Needs at least 2 days of data.

How does MoM comparison work?

It uses last month's best-saved snapshot vs. your current running total. Save to history daily for accuracy.

Can I type math in input fields?

Yes — sums like 355+2019+37609 auto-evaluate.

How do I get real AI in the assistant?

Settings tab → 6-step Cloudflare Worker setup. Takes ~10 minutes once. About $5 of credit lasts months.

What file types can I upload?

Excel (.xlsx), CSV, TSV, and text (.txt) parse fully. PNG/JPG/PDF show as preview only — copy text and use Paste & Parse for those.

How do browser notifications work?

Settings tab → Smart Reminders → Enable. Up to 3 alerts/day: end-of-shift, pace alerts, OOS lockouts.

I forgot to save yesterday — recoverable?

No. If you didn't save to history, today's data overwrote it. Enable the End-of-shift reminder in Settings to prevent this.

Does it work on phone?

Yes — mobile layout activates below 768px width. Best for quick checks; use desktop for data entry.

Setting up real AI (10 minutes)

Follow these steps once. After that, ask the ghost anything in any phrasing — it'll understand.

1

Get an Anthropic API key

Go to console.anthropic.com. Sign up with your email. Click Plans & Billing in the sidebar and add a payment method. Add $5 of credit — that's enough for months of daily use.

Then click API Keys → Create Key. Name it daily-brief. Copy the key (starts with sk-ant-...) somewhere safe — you won't see it again.

2

Create a free Cloudflare account

Go to workers.cloudflare.com. Click Sign up. Use any email — no credit card needed, free tier handles 100,000 requests/day (you'll use ~30).

3

Create your Worker

From the Cloudflare dashboard, click Workers & Pages → Create → Create Worker. Name it daily-brief-proxy. Click Deploy (just to create it). Then click Edit code in the top right.

4

Paste the proxy code

In the Cloudflare code editor, delete everything. Then copy the code from the Copy Worker code button below and paste it in. Click Deploy.
